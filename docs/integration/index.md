# Integration Documentation Index

This section documents runtime integrations that connect the accessibility checker to the live Robot Framework browser state.

This area is primarily for teams already using Browser Library, because the accessibility checks are designed to run inside an active test flow instead of reopening pages from scratch.

## Sections

- browser-library: active page/context access and state bridge behavior

## Scope

1. Session orchestration
2. Browser snapshot model
3. Tool adapter execution over one shared state
4. Preservation of login, cookies, local storage, and page context between checkpoints
